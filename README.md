BDD-Behat Test automation framework. Used Behat+Mink.
----------------

Installing additional packages
----------------
```bash
$> composer update
```

Run auto-tests:
----------------
(Linux)
```bash
$> bin/behat
```
(Windows)
```bash
$> bin/behat
```

Note:
----------------
Use PHP 7.1.12. PHP 7.2.0 causes a lot of issues with third party components.
